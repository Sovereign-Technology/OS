# sovereign-wallet

> Self-custodial sovereign wallet with AI-governed payment flows.

## Contents

| File | Role |
|------|------|
| `src/finance.html` | Wallet UI — balance display, send/receive, AI payment governance |

## Features

- Local key custody — private keys never leave the device
- AI-governed payment authorization
- Full balance and asset management
- Integrates with `sovereign-receipts` for transaction history
