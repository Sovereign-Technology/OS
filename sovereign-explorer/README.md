# sovereign-explorer

> Network index, relay management, and cross-network bridge for Sovereign.

## Contents

| File | Role |
|------|------|
| `src/search.html` | Sovereign Search — P2P network index, peer/content discovery |
| `src/relay.html` | Relay Hub — WebRTC + MQTT relay status, peer badge tracking |
| `src/bridge.html` | Network Bridge — cross-network identity bridge, pubkey/npub management |

## Features

**Search**
- Decentralized network index
- Peer and content discovery across the mesh

**Relay**
- Real-time relay health and connectivity status
- WebRTC peer count + MQTT signaling badges

**Bridge**
- Cross-network identity translation (pubkey ↔ npub)
- Click-to-copy identity chips
