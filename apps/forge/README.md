# app: forge

> Forge Platform — social feed, AI Studio, marketplace, and app builder.

`forge.html` is the Forge shell. It hosts the community hub, AI-assisted app development, and the Sovereign marketplace.

## What it does

- Unified social + builder platform shell
- Integrates `sovereign-social` (Square), `sovereign-ai` (Studio), and `sovereign-wallet` (marketplace payments)
- Full account management and security engine UI
- Requires: all `@sovereign/*` packages
